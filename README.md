# IstConnect 📍
**The Community Navigator for Istanbul Expats**

IstConnect is a full-stack platform designed to help expats and students in Istanbul discover hidden gems—from English-friendly cafes to the best Nigerian flavors in the city.

## Design & Engineering Highlights
* **Interactive Map:** Real-time location tagging using Leaflet.js.
* **Multi-Image Galleries:** Smooth horizontal-scrolling galleries for every location.
* **Personalized "Drops":** A dedicated space for users to manage their shared spots.
* **Clean UX:** A dark-themed, mobile-responsive interface focused on readability.

## Tech Stack
* **Frontend:** React.js, CSS3
* **Backend:** Node.js, Express
* **Database & Auth:** Supabase (PostgreSQL)
* **Maps:** OpenStreetMap & Leaflet

## Getting Started
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/IstConnect.git`
2. Install dependencies in both `frontend` and `backend` folders: `npm install`
3. Add your Supabase keys to `.env` files (see `.env.example`).
4. Start the project!
